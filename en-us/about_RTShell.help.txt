SHORT DESCRIPTION
	RTShell - PowerShell module for Request Tracker REST API v2

LONG DESCRIPTION
	A PowerShell module for interfacing with [Request Tracker (RT)](https://bestpractical.com/rt) 
	using its REST API v2 and API token authentication. Supports read operations, write operations 
	(tickets, replies, comments, attachments), and a response template system for standardized communications.
